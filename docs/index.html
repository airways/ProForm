ProForm Module

ProForm is an advanced form management module for ExpressionEngine 2.0, designed to make creation and management of forms easier for developers and end users.

Features

ProForm has the following features:

  • Forms fully configured in CP
  • Drag & drop form layout in CP with full EE template support for rendering forms from a single template or multiple templates
  • Mailing list opt-in
  • CAPTCHA support to help prevent spam
  • File uploads
  • Send notifications rendered from EE templates to admins and/or the submitter
  • CodeIgniter based validation including content filtering and encoding options (required, valid e-mail, strip HTML, base64 encode, etc.)
  • Plentiful hooks, allowing third party customization
  • IP address and user agent recording
  • Database Encryption
  • Preset values for use in messages such as tell a friend to prevent email spam
  • Cleanly organized OOP model
  • Shared ProLib library extending core EE and CI capabilities
